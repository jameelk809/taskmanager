# Todo App

A simple Todo app that allows you to create, manage, and track your tasks.

## Features

- Create a new task with a title, description, due date, and icon.
- Mark tasks as done or not done.
- Edit existing tasks, including changing the title, description, due date, and icon.
- Delete tasks.
- View a list of all tasks with their titles, descriptions, icons, and status.
- Scrollable list of tasks using `ListView.builder`.

## Getting Started

### Prerequisites

- Flutter: Make sure you have Flutter installed on your local development environment. You can download Flutter from the official Flutter website: [https://flutter.dev/docs/get-started/install](https://flutter.dev/docs/get-started/install)

### Installation

1. Clone the repository to your local machine using `git clone`:
   `git clone https://github.com/yourusername/todo-app.git`
2. Change to the project directory: `cd todo-app`
3. Run the app: `flutter run`


The app should now be running on your emulator or physical device.

## Usage

- Launch the app on your emulator or physical device.
- Add a new task by tapping the "Add Task" button.
- Fill in the task details, including the title, description, due date, and icon.
- Tap "Save" to create the task.
- View the list of tasks on the main screen, with their titles, descriptions, icons, and status.
- Tap on a task to view its details, including the due date and status.
- Mark a task as done or not done by swiping left and tapping the check icon.
- Delete a task by swiping it to the right on the main screen and tapping the delete icon.
